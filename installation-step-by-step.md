# Node.js 安装详细步骤（针对已下载文件）

## 情况说明
- **已下载文件**: `node-v24.14.1-x64.msi`
- **文件位置**: 桌面
- **用户问题**: 不知道如何打开.msi文件

## 详细步骤

### 第一步：找到并打开安装文件
1. **回到桌面**：
   - 按键盘上的 `Windows键 + D`（同时按）可以直接回到桌面
   - 或者点击屏幕右下角时间旁边的"显示桌面"小按钮

2. **找到安装文件**：
   - 在桌面上找到名为 `node-v24.14.1-x64.msi` 的文件
   - 图标应该是一个蓝色的盒子，上面有白色"npm"字样

3. **双击打开文件**：
   - 用鼠标左键快速点击文件两次（双击）
   - 如果双击没反应，可以：
     a. 右键点击文件
     b. 选择"打开"或"以管理员身份运行"

### 第二步：处理可能的系统提示
**可能出现的情况1：用户账户控制提示**
```
用户账户控制
你要允许此应用对你的设备进行更改吗？
```
- **操作**：点击"是"（Yes）

**可能出现的情况2：Windows保护你的电脑**
```
Windows已保护你的电脑
Microsoft Defender SmartScreen阻止了无法识别的应用启动...
```
- **操作**：点击"更多信息"，然后点击"仍要运行"

### 第三步：安装过程
安装程序打开后，按照以下步骤操作：

1. **欢迎界面**：
   ```
   Node.js Setup
   Welcome to the Node.js Setup Wizard
   ```
   - 点击"Next"（下一步）

2. **许可协议**：
   ```
   End-User License Agreement
   ```
   - 勾选"I accept the terms in the License Agreement"（我接受许可协议中的条款）
   - 点击"Next"

3. **安装路径**：
   ```
   Destination Folder
   ```
   - **不要修改**，使用默认路径：`C:\Program Files\nodejs\`
   - 点击"Next"

4. **重要：自定义设置**：
   ```
   Custom Setup
   ```
   - 确保勾选了 **"Automatically install the necessary tools..."**（自动安装必要工具）
   - **必须勾选**：`Add to PATH`（添加到PATH环境变量）
   - 其他选项保持默认
   - 点击"Next"

5. **准备安装**：
   ```
   Ready to install Node.js
   ```
   - 点击"Install"（安装）

6. **安装进行中**：
   - 等待进度条完成（约30秒-2分钟）
   - 可能需要管理员权限，点击"是"确认

7. **安装完成**：
   ```
   Completed the Node.js Setup Wizard
   ```
   - 点击"Finish"（完成）

### 第四步：验证安装
安装完成后，需要验证是否成功：

1. **打开命令提示符**：
   - 按键盘 `Windows键 + R`
   - 输入 `cmd` 然后按回车
   - 会打开一个黑色的命令行窗口

2. **输入验证命令**：
   在黑色窗口中输入以下命令，每个输入后按回车：

   ```bash
   node --version
   ```
   **期望看到**：`v24.14.1`（或类似版本号）

   ```bash
   npm --version
   ```
   **期望看到**：`10.9.0`（或类似版本号）

   ```bash
   npx --version
   ```
   **期望看到**：`10.9.0`（或类似版本号）

### 第五步：常见问题解决

#### 问题1：双击没反应
- **解决方案**：
  1. 右键点击文件
  2. 选择"以管理员身份运行"
  3. 如果还是没反应，检查文件是否损坏，重新下载

#### 问题2：'node'不是内部或外部命令
- **原因**：PATH环境变量未正确设置
- **解决方案**：
  1. 重新运行安装程序
  2. 确保勾选了"Add to PATH"
  3. 安装完成后，重启电脑，然后重试验证命令

#### 问题3：安装被拦截
- **解决方案**：
  1. 暂时关闭杀毒软件
  2. 右键"以管理员身份运行"
  3. 允许所有系统提示

### 安装完成标志
✅ **成功标志**：
- 三个验证命令都显示版本号
- 没有错误信息

❌ **失败标志**：
- 显示"不是内部或外部命令"
- 安装过程中有红色错误提示
